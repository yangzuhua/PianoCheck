# 钢琴打卡 · 家长监督 APP

一个**只有家长端**的安卓 APP，用于监督小孩练琴并自动打卡记录。

## 功能

1. **家长自定义设置**
   - 练习总时长（倒计时，如 30 分钟）
   - 静音多少秒后提醒（如 30 秒没听到钢琴声就提醒）
   - 麦克风灵敏度（环境越安静可设越小）
   - **静音提醒文案**（自定义，如「宝贝，继续弹琴哦，不要停~」）
   - **完成鼓励文案**（自定义，如「今天弹得真棒，继续保持！」）

2. **实时监听 + 语音提醒**
   - APP 通过麦克风实时监听环境声音，检测到「连续 N 秒低于阈值」即判定为没在认真弹，
     自动用 TTS 语音播报你设置的提醒文案。
   - 倒计时结束 / 手动结束时：若**全程没有被提醒** → 播报鼓励文案，并记录为「弹得好」；
     若**有过提醒** → 记录为「有提醒 · 没认真弹」。
   - 锁屏后仍在后台监听（前台服务 + 通知），不中断。

3. **打卡记录**
   - 每次练习自动存档：日期时间、计划时长、实际弹奏秒数、提醒次数、状态（弹得好 / 有提醒·没认真弹）。
   - 「记录」页按时间倒序展示，可删除单条或清空。
   - 所有数据**仅保存在本机**，不上传任何服务器，保护隐私。

> 说明：v1 用「声音能量（音量）阈值」判断是否在弹琴，对钢琴声 / 其他较大声音一视同仁。
> 这足以区分「在弹」与「停下了 / 去玩了」。后续可升级为 AI 识别钢琴音色。

## 环境要求

- **Android Studio** （建议 Hedgehog / Iguana 或更新版本）
- **JDK 17**
- 一台 **Android 7.0（API 24）及以上** 的手机（用于安装运行）

---

## 方式一：用 Android Studio 构建 APK（推荐，最简单）

1. 把本目录 `PianoCheck` 整个拷贝到电脑。
2. 打开 **Android Studio → File → Open**，选择 `PianoCheck` 文件夹，等待 Gradle 同步完成
   （首次会下载依赖，需要联网，约几分钟）。
3. 连接手机（开启「开发者选项 → USB 调试」），或新建一个模拟器。
4. 顶部菜单 **Build → Build Bundle(s) / APK(s) → Build APK(s)**。
5. 编译完成后右下角会弹出提示，点 **locate** 找到生成的
   `app/build/outputs/apk/debug/app-debug.apk`（或 release 版）。
6. 把这个 `.apk` 传到手机安装即可（安装未知来源应用请按手机提示授权）。

> 想生成**正式签名**的 release APK：Build → Generate Signed Bundle / APK，按向导创建/选择你的密钥。

---

## 方式二：GitHub Actions 云打包（推荐，免装 Android Studio）

仓库已附带 `.github/workflows/build-apk.yml`。把代码推到 GitHub 后，在仓库
**Actions** 页面手动运行 `Build APK` 工作流，结束后即可在 Artifacts 中下载 APK
（产物名 `piano-check-apk`，里面是 `app-release.apk`）。

> 工作流会**自动安装 JDK 17、Android SDK 与 Gradle 8.9** 并直接构建，
> 因此**不需要**仓库里提供 `gradle-wrapper.jar`（本地用 Android Studio 打开才会生成它）。

## 使用流程

1. 打开 APP，进入「**设置**」页，按需要填写时长、静音阈值、提醒/鼓励文案，点「保存设置」。
2. 切到「**监督**」页，点「**开始监督**」，首次会请求麦克风 / 通知权限，请允许。
3. 让孩子开始弹琴。APP 开始倒计时并显示实时声音条。
   - 中途停下超过设定的秒数 → 手机自动语音提醒。
   - 练完（倒计时归零或点「结束」）→ 没被提醒就语音鼓励，并写入「记录」。
4. 随时到「**记录**」页查看每天的打卡情况。

## 灵敏度调参建议

- 在安静房间、钢琴正常音量下，灵敏度保持默认 `0.02` 一般可用。
- 若把电视声、家长说话声也误判成「在弹琴」（即该提醒时没提醒），把**灵敏度调大**（如 0.05）。
- 若孩子正常弹琴却被频繁误提醒，把**灵敏度调小**（如 0.01）。
- 也可调大「静音多少秒后提醒」，避免短暂停顿触发。

## 隐私

所有设置与打卡记录只存在手机本地数据库（DataStore + Room），APP 不联网、不上传任何数据。

## 已知限制 / 后续可加

- v1 用音量阈值判断，不能区分「钢琴声」与「其他声音」。
- 当前为单个小孩；后续可加多小孩切换。
- 可增加日历视图、每周统计、导出 CSV、自定义提醒语音（录音）等。
